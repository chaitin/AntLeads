What is AntLeads Widget?

AntLeads Widget is an embeddable JavaScript widget that enables you to collect leads from any website. With just a single line of code, you can add a beautiful, customizable contact form that appears as a floating button or can be triggered from any element on your page.

All submissions are automatically scored by our AI engine, tagged, and added to your lead pipeline with intelligent routing and follow-up automation.

How to Integrate

Create a Widget

Log into your AntLeads dashboard, go to the Widgets page, and create a new widget. Customize the appearance, form fields, and behavior to match your needs.

Get Your Embed Code

After creating the widget, copy the provided embed code. It looks like this:

<script src="http://localhost:8000/static/widget.js"
        data-widget-id="wgt_YOUR_WIDGET_ID"></script>

Add to Your Website

Paste the embed code into your website's HTML, just before the closing </body> tag. The widget will automatically appear as a floating button.

Start Collecting Leads!

That's it! Your widget is now live. All form submissions will appear in your AntLeads dashboard with AI scoring and automatic follow-up tasks.

Advanced Usage

Bind to Custom Element

You can bind the widget to any button or div on your page:

<button id="contact-button">Contact Us</button>

<script src="http://localhost:8000/static/widget.js"
        data-widget-id="wgt_YOUR_WIDGET_ID"
        data-bind-to="#contact-button"></script>

Programmatic Control

Control the widget programmatically using the JavaScript API:

// Open the widget
window.AntLeadsWidget.open();

// Close the widget
window.AntLeadsWidget.close();

// Check if widget is open
if (window.AntLeadsWidget.isOpen()) {
  console.log('Widget is currently open');
}
